# PiggyBank - Blockchain

### Requirements

- Node v12.21.0

### Installation

```
git clone https://github.com/anonimowy891/piggybank
cd collectibles/blockchain/
npm i
```

### Usage

#### Start Node

```
./bin/run start --enable-http-api-plugin --http-api-plugin-port 4000  --http-api-plugin-whitelist 0.0.0.0/0
```
